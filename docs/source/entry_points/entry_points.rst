Entry Points
============

e2xgrader supports a flexible plugin system using Python entry points, allowing you to extend and customize its behavior for different cell types. The two main extension points are **Cell Filters** and **Cell Preprocessors**.

This page provides an overview of both systems and links to detailed documentation for each.
